# Api de automação de interações com a LexisNexis

## Instruções de uso
Instalar pacotes node - `npm i`
Executar projeto - `npm start`

## Teste
Alterar configuration.js para module.exports = configsDev, executar projeto e abrir no navegador 
http://localhost:8000/download?busName=Equatorial&busCountry=Brazil
